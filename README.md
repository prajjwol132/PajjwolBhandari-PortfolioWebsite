# Prajjwol Bhandari - Portfolio Website

A modern, responsive portfolio website showcasing my skills as a Web Developer, React Developer, Video Editor, and Creative Designer.

![Portfolio Preview](profile.png)

## Features

- **Responsive Design** - Fully responsive layout that works on all devices
- **Modern UI/UX** - Clean and professional design with smooth animations
- **Scroll Progress Bar** - Visual indicator of page scroll progress
- **Typing Animation** - Dynamic text animation showcasing different roles
- **Smooth Scrolling** - Seamless navigation between sections
- **AOS Animations** - Beautiful scroll-triggered animations
- **Floating Shapes Background** - Decorative animated background elements

## Sections

1. **Home/Hero** - Introduction with profile image and call-to-action buttons
2. **About** - Personal information and background
3. **Skills** - Technical skills and expertise
4. **Projects** - Portfolio of work samples
5. **Contact** - Contact form and social links

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Custom styling with CSS variables and gradients
- **JavaScript** - Interactive features and animations
- **Bootstrap 5.3.2** - Responsive grid and components
- **Font Awesome 6.4.0** - Icon library
- **AOS (Animate On Scroll) 2.3.1** - Scroll animations

2. Open `index.html` in your browser
3. No build process or dependencies to install

## Customization

### Changing Colors
Edit the CSS variables in [`style.css`](style.css:1):
```css
:root {
    --primary: #F0BE6F;
    --secondary: #1B4543;
    --accent: #B3D0C6;
    --dark: #0a1f1d;
    --light: #f8faf9;
}
```

### Changing Typing Text
Edit the text array in [`script.js`](script.js:36):
```javascript
const textArray = ['Web Developer', 'React Developer', 'Video Editor', 'Creative Designer'];
```
